Video Quality Guide

  • 4K (2160p): Ultra HD quality, 3840x2160 pixels. File size: 1-3 GB/hour. Best for large screens and professional use.
  • 2K (1440p): Quad HD quality, 2560x1440 pixels. File size: 700 MB-1.5 GB/hour. Great for gaming monitors.
  • Full HD (1080p): High definition, 1920x1080 pixels. File size: 400-800 MB/hour. Perfect for most users - recommended!
  • HD (720p): Standard HD, 1280x720 pixels. File size: 200-400 MB/hour. Good for mobile devices.
  • SD (480p): Standard definition, 854x480 pixels. File size: 100-200 MB/hour. Saves storage space.
  • Low (360p): Low quality, 640x360 pixels. File size: 50-100 MB/hour. For slow connections only.

Audio Quality Guide

  • 320 kbps: Maximum MP3 quality. File size: 3-4 MB/song. Virtually indistinguishable from CD quality - recommended for music!
  • 256 kbps: High quality. File size: 2.5-3 MB/song. Excellent for most listening scenarios.
  • 192 kbps: Good quality. File size: 2 MB/song. Noticeable compression on high-end audio systems.
  • 128 kbps: Standard quality. File size: 1.5 MB/song. Suitable for voice content and podcasts.
