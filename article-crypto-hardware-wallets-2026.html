Hardware Wallet Setup Guide

1

Buy From Official Sources Only

Never buy from Amazon third-party sellers or eBay. Only buy from ledger.com or trezor.io. Pre-loaded seed phrases = scam devices.

2

Unbox and Verify Packaging

Check for tamper-evident seals. Ledger has a security checklist at ledger.com/start. Device should show "Welcome" on first boot.

3

Generate Your Seed Phrase

Device will generate 24 random words. Write them down ON PAPER. Never type them digitally. Never screenshot. Never email.

4

Verify Seed Phrase

Device will ask you to confirm random words. This ensures you wrote them correctly. If you fail, start over.

5

Set PIN Code

Choose a strong 6-8 digit PIN. After 3 wrong attempts, device wipes. This protects against physical theft.

6

Install Apps & Transfer Funds

Use Ledger Live or Trezor Suite to install coin apps. Send a small test transaction first. Verify receiving address on device screen.

05

Security Best Practices

Seed Phrase Storage

Store on metal (fire/water resistant). Keep in secure location. Consider multiple copies in different locations. Never store digitally.

Never Share Your Seed

No support agent will ever ask for it. No airdrop requires it. No "wallet sync" needs it. Anyone asking = scam. Period.

Verify Addresses on Device

Always confirm the receiving address shown on your hardware wallet screen matches. Malware can change clipboard addresses.

Keep Firmware Updated

Update through official apps only (Ledger Live/Trezor Suite). Updates patch security vulnerabilities.
